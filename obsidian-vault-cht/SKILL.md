# Obsidian Vault

協助使用者在 Obsidian vault 中搜尋、建立與整理筆記。

## 工作流程

1. 找到 vault root。若使用者未提供，從目前工作目錄與常見位置推斷；找不到再詢問。
2. 使用全文搜尋找既有相關筆記，避免重複建立。
3. 建立新筆記時使用清楚標題、frontmatter 或既有格式。
4. 使用 [[wikilinks]] 連到相關筆記。
5. 若有 index note 或 MOC，更新索引。
6. 保留使用者既有命名與資料夾慣例。

## 原則

- 不要在沒有檢查既有筆記前建立重複內容。
- 不要把 Obsidian 筆記改成過度正式文件；保持個人知識庫風格。
- 不要刪除或重排大量筆記，除非使用者明確要求。
- 對不確定的分類，提出建議並說明理由。
